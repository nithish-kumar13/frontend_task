## Tech Stack

**Client:** React.js and Tailwind CSS

**Server:** Node, Express


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn run start
```
